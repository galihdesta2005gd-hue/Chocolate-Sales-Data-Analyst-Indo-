# International Chocolate Sales Performance Dashboard
Sebuah proyek analisis data dan intelijen bisnis komprehensif yang berfokus pada pelacakan kinerja penjualan internasional, efisiensi rantai pasok, dan profitabilitas produk untuk distributor cokelat global.

# Business Impact & Key Insights
- **Pendorong Pendapatan:** Mengidentifikasi produk cokelat dengan performa terbaik dan pasar regional yang berkontribusi sebesar **[Insert % or total amount]** dari total pendapatan.
- **Keuntungan Analysis:** Menunjukkan produk-produk dengan margin rendah dan mengoptimalkan tren penjualan untuk membantu para pemangku kepentingan meningkatkan margin keuntungan secara keseluruhan.
- **Effisiensi Metrics:** Mengevaluasi frekuensi transaksi dan waktu pengiriman pengapalan untuk menyoroti hambatan (bottleneck) pada rantai pasok.

# Tools & Technologies Used
- **Excel / Python (Pandas):** Untuk pembersihan data, penanganan nilai yang hilang (missing values), dan rekayasa metrik tanggal/penjualan.
- **Power BI** Untuk pemodelan data, pembuatan visual interaktif, dan perancangan antarmuka pelaporan tingkat eksekutif.

# Project Structure
- `chocolate_sales_data.csv`: Dataset transaksi yang telah dibersihkan.
- `Chocolate Sales Data Dashboard`: File dashboard Power BI yang berisi model data dan lembar interaktif.
- `README.md`: Dokumentasi proyek dan ringkasan eksekutif.

# Key Metrics Tracked
1. **Total Penjualan & Profit Margin:** KPI eksekutif tingkat tinggi untuk memantau kesehatan finansial.
2. **Penjualan dari Negara/Daerah:** Analisis geospasial untuk menemukan pasar yang kurang berkinerja.
3. **Product Performance:** Rincian jenis cokelat terlaris versus inventaris yang lambat terjual.
